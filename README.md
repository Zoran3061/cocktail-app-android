# Cocktail App (Android • Kotlin)

An Android mobile application built with **Kotlin** that allows users to browse cocktails and view detailed information through a clean and user-friendly interface.

## Features
- Browse a list of cocktails
- View cocktail details
- Simple navigation and clean UI
- Optimized for mobile devices

## Tech Stack
- Kotlin
- Android SDK
- Android Studio
- Gradle

## Project Structure
- `app/` – application source code (UI and logic)
- `gradle/` – Gradle wrapper and configuration files

## Getting Started (Run Locally)
1. Clone the repository
   ```bash
   git clone https://github.com/Zoran3061/cocktail-app-android.git
   ```
2. Open the project in Android Studio
3. Wait for Gradle Sync to complete
4. Run the application on an Android emulator or a physical device
